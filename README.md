# DiscordBot
Building a Discord bot with Go

### Functionality
We will ultimately want a handful of commands and features for the bot.

#### Organize Calls
- The bot should open up a post to allow people to react to a post indicating that they
want to join a scheduled call.
- The command should be `!botname - call - day - time - topic`
- The bot should @ people who reacted indicating that they should join some time prior to the call (5 minutes)


#### Tell Jokes
- Simple tell a joke function
